### About NotifyMe

**NotifyMe** is a Discord Bot being used by 5k+ Discord Servers, that allows you to notify your Discord Community, whenever your favorite content creators post on YouTube, or go live on Twitch and Kick.com!

I decided to build this Discord Bot, because I felt like there wasn't a good notifications bot available, that not only was easy-to-use, but also had enough free channels, so here we are!
